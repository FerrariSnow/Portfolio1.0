# Portfólio 1.0

Este repositório contém o código-fonte para o meu primeiro portfólio de projetos front-end. O portfólio consiste em uma lista de projetos, onde cada item leva para um projeto diferente. O portfólio pode ser acessado online em [https://portfolio1-0-jet.vercel.app/](https://portfolio1-0-jet.vercel.app/).

## Tecnologias Utilizadas

O portfólio foi desenvolvido utilizando as seguintes tecnologias:

- HTML
- CSS ( LESS )
